# GO DATA TYPE 

### DATA TYPE DEFINITION
--> Data types are the classifications given to different kinds of values used in everyday programming because it is an important concept and it specifies the size and type of variable values.

### TYPES OF DATA TYPE

***In Go there are seven basic data types but here are three***

--> Bool ==> This represents a boolean value and it's either true or false.

--> Numeric ==> This represents integer types, floating point values and complex types.

--> String ==> This represents a string value
