# GO CONSTANTS

### CONSTANT 

--> constant which means unchangeable are declared outside the functions especially at the package level. Constants are immutable they cannot be changed during the program execution so you can't modify its value. 
the keyword for constant is "const"

### TYPES OF CONSTANT 

--> Typed constant ==> which are declared with a defined type

--> Untyped constant ==> which are declared without a type 

**NOTE** ==> Constant can be declared both inside and outside of a function and the names are usually written in uppercase letters, they also follow the same rules as variables.
