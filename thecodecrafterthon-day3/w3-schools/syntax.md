# GO SYNTAX

The basics syntax in Go are :  

--> The package declaration 

--> Import packages 

--> The functions 

--> Statement and expressions 

## A BASIC STRUCTURE OF GO PROGRAM 

package main // this is standalone program and an executable program that shows that it belongs to the main package. 

import "fmt" //   we are importing the fmt which is a standard package from the standard library.

func main() { // this is starting point of a program and any code inside the curly brackets executable and it also makes it readable.
    fmt.Println("Hello, World!") // the fmt.Println is an available functions from the fmt package which is use to print text and "Hello, World!" is a statement and statements. And it's executable because its inside the curly bracket.
}
