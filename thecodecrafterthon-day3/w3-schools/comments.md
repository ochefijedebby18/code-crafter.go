# GO COMMENTS

## COMMENTS

--> Comments are used to explain what the code does and why something was done a certain way. There are for outlining important or fragile blocks of code that require extra code. There are also used for disabling code without deleting it. 

## TYPES OF COMMENTS

--> Line comments(single lines)

--> Block comments(multi lines)

### Line comments(single lines) 

--> The line comments start with // and the rest of the line is ignored by the compiler.

###  Block comments(multi lines)

-- > The block comments can scan multiple lines and it start with /* and ends with */ enveloping everything inside.
