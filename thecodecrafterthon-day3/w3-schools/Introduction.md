# INTRODUCTION TO GO

## WHAT IS GO 

--> Go is a programming language created at google, designed by Robert Griesemer, Rob Pike and Ken Thompson. That process began in september 2007 and was publicly announced as an open source project in 2009. It was designed to be simple, fast and powerful. 

##  WHAT GO IS USED FOR 

--> Go is used for backends.

--> Go is used to build web servers, APIs and also cloud systems.

--> Go is used also for large scalable systems.


## WHY WE USE GO 

--> We use Go because of its simplicity so it makes it easier for a new developer to learn and understand it faster. In Go code is readable and the bugs can easily be fix.
